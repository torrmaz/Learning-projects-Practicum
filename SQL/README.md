## Цель проекта
Ваша первая задача как аналитика — проанализировать базу данных. В ней — информация о книгах, издательствах, авторах, а также пользовательские обзоры книг. Эти данные помогут сформулировать ценностное предложение для нового продукта.
## Инструменты
Проект делал с помощью Python в Jupyter Notebook. Использовал библиотеки pandas, sqlalchemy 
## Решение
Посчитано, сколько книг вышло после 1 января 2000 года;

Для каждой книги посчитано количество обзоров и среднюю оценку

Определено издательство, которое выпустило наибольшее число книг толще 50 страниц — так мы исключаем из анализа брошюры;

Определен автор с самой высокой средней оценкой книг — учтены только книги с 50 и более оценками;

Посчитайтано среднее количество обзоров от пользователей, которые поставили больше 48 оценок;
